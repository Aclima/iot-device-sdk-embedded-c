Google Cloud IoT Device SDK for Embedded C API

Connects client applications to Cloud IoT CoreThe Device SDK securely connects client applications to Cloud IoT Core in order to publish and subscribe to messages via MQTT.

Getting started

To build the Device SDK:

  1. git clone https://github.com/GoogleCloudPlatform/iot-device-sdk-embedded-c.git
  2. cd iot-device-sdk-embedded-c
  3. (Optional) Customize the Board Support Package for your device.
  4. make

Dependencies

The Device SDK calls the hardware-specific drivers and routines in the Board Support Package (BSP) to implement the MQTT protocol. The BSP is a part of the Device SDK.

The Device SDK has a turn-key POSIX BSP, so it builds natively on POSIX platforms. Customize the BSP to build the Device SDK on non-POSIX platforms.
